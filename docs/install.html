Installation

Python 3

Python 3.7 or greater is required to run Firmetix. You can download it from the official python site here.

Verify python install

To check that you have the correct version of Python 3 installed, open a command window and type:

python3 -V

Windows, you may need to type:

python -V

The Python version will be displayed: 

python3
Python 3.7.2 (default, Dec 31 2018, 14:25:33)
[GCC 8.2.0] on linux
Type "help", "copyright", "credits" or "license" for more information.
>>>

For Windows users, this may look something like:

python
C:\Users\user_name>python
Python 3.9.0 (tags/v3.9.0:9cf6752, Oct  5 2020, 15:34:40) [MSC v.1927 64 bit (AMD64)] on win32
Type "help", "copyright", "credits" or "license" for more information.
>>>

Installing Firmetix

To install Firmetix, open a command window and type:

pip3 install firmetix

Windows, you may need to type:

pip install firmetix

This will install Firmetix and all of its dependencies.

The code is also available in this GitHub repo

Installing Firmetix4Arduino

In the arduino IDE go in the Manage Libraries panel and search for Firmetix4Arduino

Install the library and you are ready to go

Open the example script and upload it to the arduino

You can change the ARDUINO_ID at the top of the script to have multiple arduinos

If your missing memory you can comment out,with //, one or multiple of #define Enable

The code is also available in this GitHub repo
